# Calendrier-Assignment Tracker
*Project Overview*
Calendrier is an assignment tracker build with the intention of helping young students to keep track of their assginments to improve their academic perfomance and decrease their feeling of overwhelm.
We build a web application that uses FlaskAPI, Python, CSS and database management tools like SQLAlchemy to complete a multilayered tool that saves your assignments and also allows you to connect your D2L Calendar.
You can:
- Add, Update and Delete assginments
- Filter by class
- Save your information with an Account functionality
- Study with the Pomodoro timer
- Have a montly view of the assignmets due

*Our Vision*
Help students to succed by eliminating clutter, overwhelming unorganized due dates and avoid procrastination with an clean and easy-to-use application build for students.
